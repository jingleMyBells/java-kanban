# java-kanban
Таск-трекер v0.0.1  
Реализованы базовые классы задач, эпиков и подзадач
Реализован менеджер задач и методы его работы с сущностями трекера

## Техстэк

Java 11

# Запуск

Сбилдить исходники:
```bash
  javac -encoding UTF Main.java
```
Запустить:
```bash
  java -Dfile.encoding=UTF-8 Main
```
